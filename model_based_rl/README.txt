Versiones dependencias:

- numpy: 1.19.3
- matplotlib: 3.4.2
- torch: 2.0.0
- pandas: 1.2.4
- gym: 0.23.1

Instrucciones para ejecutar el código:

- Abra una terminal bash
- Ejecute el comando "cd" seguido por el path hacia la carpeta donde se encuentra el archivo run.sh
- Ejecute el comando "sh run.sh"

El script generate_plots.py contiene el código necesario para generar los gráficos de los experimentos de esta tarea a partir
de los archivos .csv generados a partir del script run.sh
